Projet: Who's Jack? 
Auteurs: Groupe 6SD - Le Stum Sébastien - Peytoureau Julie - Ravelonanosy Lova - Tran Anthony

Ce projet a été proposé par Mr Denis Bureau dans le cadre de l'apprentissage de programmation Java. 
Ce projet est un jeu d'aventure, inspiré du jeu zuul-bad.

Il existe deux modes de lancement du jeu:
	- soit par l'intermédiaire d'un applet : voir http://www.esiee.fr/~tanth/ ou http://www.esiee.fr/~lestuse/
	- soit par le terminal: aller dans le dossier contenant le fichier whoisjack.jar et taper la commande java -jar whoisjack.jar

Cette version contient plusieurs paquetages et plusieurs classes dont la principale est la classe Game.

Les images ont été conçues par Anthony Tran. Les sites web contiennent le scénario, le rapport, et les javadocs du jeu.
